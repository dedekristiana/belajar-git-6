kita voba lagi membuat sebuah file baru menggunakan git. pertama kita buat folder lalu buka dengan git bash, lalu kita buat file menggunakan visual code setelah file kita buat ketikan perintah git status pada git dan lihat announce yang ada di git apakah teradapat file baru, jika iya maka kita simpan menggunakan perintah git add . lalu coba kita ketikan perintah git status dan lihat apakah sudah berbah announce nya ??

jika sudah berubah announce nya kiba ke tahap selanjutnya membuat sebuah pesan file menggunakan perintah git commit -m "(di isi sesuai apa yg di buat atau revisi)" setelah membuat sebuah commit kita buat sebuah repositor di dalam git hub lalu salin alamat web yang muncul. kita kembali lagi ke dalam git lalu kita ketikan perintah git remote add origin (di tambah url yang taadi di salin) tujuannya agar projek kita terhubung dengan git hub. setelah kita hubungan dengan git hub.. kita unggal perubahan yang file yang sudah kita buat dengan perintah git push -u origin master
